\section{Εργαλεία Και Τεχνικές}

Για την υλοποίηση του REST API χρησιμοποιήθηκε το framework\footnotemark\footnotetext{Software framework, a reusable set of libraries or classes for a software system } Lumen \cite{DUMMY:1} που ενδείκνυται για REST API. Ωστόσο τα προαπαιτούμενα για την χρήση του framework είναι:  

\begin{itemize}
\item PHP >= 5.6.4
\item OpenSSL PHP Extension
\item PDO PHP Extension
\item Mbstring PHP Extension
\end{itemize}

Τα οποία ήταν διαφορετικά στο προσωπικό υπολογιστή μου (PHP 5.61) και για το λόγο αυτό χρησιμοποιήθηκε το εικονικό περιβάλλον homstead με τη χρήση VirtualBox και Vagrant.
